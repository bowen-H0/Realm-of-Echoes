插件物品创建指南

按照以下步骤创建插件物品：

  1. 在项目中创建两个文件夹：A 和 B。
  2. 在 B/item/ 文件夹中创建物品的 json 文件（文件名：ID+.json）。
  3. 将物品贴图放置在 A/item/ 文件夹中，贴图文件名应与 json 文件的 ID 相同（文件名：ID+.png）。

示例 1：创建强化铁锭


{
    "ID": "Enhanced-Iron-Ingot",
    "type": "items",
    "description": "A special type of iron, stronger and more durable than regular iron.",
    "mod": "Synthesis_table",
    "table": {
        "Type": "items",
        "items": {
            "Iron-Ingot": {
                "Num": 2
            },
            "BlueSlime-ball": {
                "Num": 1
            }
        }
    }
}

示例 2：创建水晶碎片


{
    "ID": "Crystal-Shard",
    "type": "Items",
    "description": "A sparkling shard of crystal, imbued with mystical energies. It is a key ingredient for crafting powerful weapons and enhancing gear.",
    "mod": "Drop",
    "table": {
        "AzureSkeleton": [
            {"scene": "Crystal-Shard", "rarity": 10}
        ],
        "MysticStoneSentinel": [
            {"scene": "Crystal-Shard", "rarity": 17}
        ]
    }
}

示例 3：创建星光剑


{
    "ID": "Starlight",
    "type": "Weapons",
    "description": "A gorgeous and dazzling glowing sword, known for its agility and speed.",
    "mod": "Synthesis_table",
    "table": {
        "Type": "Weapons",
        "items": {
            "Sapphire": {
                "Num": 1
            },
            "Iron-Ingot": {
                "Num": 1
            },
            "Wood": {
                "Num": 1
            }
        }
    },
    "attack": {
        "Sharp": "(1,1)",
        "Fast": "(1,3)"
    }
}
