插件物品创建指南

按照以下步骤创建插件物品：

  1. 在项目中创建两个文件夹：A 和 B。
    A 文件夹用于存放物品的贴图，B 文件夹用于存放物品的配置文件。
  2. 在 B/item/ 文件夹中创建物品的 json 文件（文件名：ID+.json）。
  3. 将物品贴图放置在 A/item/ 文件夹中，贴图文件名应与 json 文件的 ID 相同（文件名：ID+.png）。

示例 1：创建强化铁锭

这个示例展示了如何创建一个叫做“强化铁锭”的物品，它是通过合成两块铁锭和一颗蓝史莱姆球制作出来的。


{
    "ID": "Enhanced-Iron-Ingot",  
    "type": "items",  
    "description": "A special type of iron, stronger and more durable than regular iron.",  
    "mod": "Synthesis_table",  
    "table": {  
        "Type": "items",  
        "items": {  
            "Iron-Ingot": {
                "Num": 2  
            },
            "BlueSlime-ball": {
                "Num": 1  
            }
        }
    }
}

示例 2：创建水晶碎片

这个示例展示了如何创建一个叫做“水晶碎片”的物品，它可以通过打怪掉落，掉落的怪物包括 AzureSkeleton 和 MysticStoneSentinel。


{
    "ID": "Crystal-Shard",  
    "type": "Items",  
    "description": "A sparkling shard of crystal, imbued with mystical energies. It is a key ingredient for crafting powerful weapons and enhancing gear.",  
    "mod": "Drop",  
    "table": {  
        "AzureSkeleton": [
            {"scene": "Crystal-Shard", "rarity": 10}  
        ],
        "MysticStoneSentinel": [
            {"scene": "Crystal-Shard", "rarity": 17}  
        ]
    }
}

示例 3：创建星光剑

这个示例展示了如何创建一个叫做“星光剑”的武器，它由蓝宝石、铁锭和木材合成，并且具有额外的攻击属性。


{
    "ID": "Starlight",  
    "type": "Weapons",  
    "description": "A gorgeous and dazzling glowing sword, known for its agility and speed.",  
    "mod": "Synthesis_table",  
    "table": {  
        "Type": "Weapons",  
        "items": {  
            "Sapphire": {
                "Num": 1  
            },
            "Iron-Ingot": {
                "Num": 1  
            },
            "Wood": {
                "Num": 1  
            }
        }
    },
    "attack": {  
        "Sharp": "(1,1)",  
        "Fast": "(1,3)"  
    }
}

示例 4：创建星雾剑

这个示例展示了如何通过升级星光剑来创建一个新的武器“星雾剑”。


{
    "ID": "Star-Mist",  
    "type": "Weapons",  
    "description": "The blade is like starlight, gleaming with a faint glow, light and formless. It swings like the wind, silent in its deadly intent.",  
    "mod": "WeaponsUpgrade",  
    "table": {  
        "Weapon": "Starlight",  
        "items": {  
            "Platinum-Ingot": {
                "Num": 1  
            },
            "Enhanced-Iron-Ingot": {
                "Num": 2  
            },
            "Crystal-Shard": {
                "Num": 2  
            }
        }
    },
    "attack": {  
        "Sharp": "(1,1)",  
        "Fast": "(1,5)",  
        "Undead_killer": "(1,1)"  
    }
}

插件配置示例

然后我们在 plugin.json 的 configuration 中的 item 添加你新物品的ID。


{
    "plugin_name": "Sword Extension",
    "version": 1.1,
    "author": "Realm of Echoes",
    "description": "Core extension",
    "configuration": {
        "item": [
            "Enhanced-Iron-Ingot",
            "Enhanced-Iron-Sword",
            "Geargrinder",
            "Star-Mist",
            "Unyielding-Forged-Blade",
            "Starlight",
            "Pure-Gold-Sword",
            "Crystal-Shard",
            "Platinum-Sword-II",
            "Chrome-longsword-II"
        ]
    }
}

完成配置后，你可以将插件安装到游戏中，并在 选项 > 扩展 页面查看。
