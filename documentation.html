安装指南

请按照以下步骤安装插件：

  1. 下载插件文件。
  2. 打开游戏，点击 Option > Extensions > Managing plugins，这会自动用文件管理器打开插件目录。
  3. 将插件文件解压到游戏插件目录。
  4. 重启游戏，插件将自动生效。

开发指南

如果您想要开发自己的插件，以下是一些基础步骤和示例：

  1. 创建一个插件文件夹，命名为您插件的名称（可以和插件名称不一样）。
  2. 在插件文件夹中创建一个 plugin.json 文件，放在项目根目录。
  3. 配置 plugin.json 文件的基本结构，如下所示：

{
    "plugin_name": "name",
    "version": 1,
    "author": "by",
    "description": "description",
    "configuration": {
    }
}

在 plugin.json 文件中，您可以为插件设置名称、版本、作者、描述和配置项。确保文件格式正确，且包含必要的信息。

现在我们就制作一个插件了，在游戏安装它，它应该就能显示在游戏Option > Extensions的插件列表

示例


{
    "plugin_name": "Sword Extension",
    "version": 1.1,
    "author": "Realm of Echoes",
    "description": "Core extension",
    "configuration": {
        "item":[
            "Enhanced-Iron-Ingot",
            "Enhanced-Iron-Sword",
            "Geargrinder",
            "Star-Mist",
            "Unyielding-Forged-Blade",
            "Starlight",
            "Pure-Gold-Sword",
            "Crystal-Shard",
            "Platinum-Sword-II",
            "Chrome-longsword-II"
        ]
    }
}

我们可以在 configuration 里注册插件包含的功能，并且我们可以让一个插件的 configuration 同时有 object 和 item。
