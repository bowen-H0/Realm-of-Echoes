安装指南

请按照以下步骤安装插件：

  1. 下载插件文件。
  2. 打开游戏，点击 Option > Extensions > Managing plugins，这将自动打开文件管理器并定位到插件目录。
  3. 将下载的插件文件解压到游戏插件目录。
  4. 重启游戏，插件将在游戏中自动生效。

完成这些步骤后，您就可以在游戏的插件管理页面查看并启用已安装的插件。

开发指南

如果您希望开发自己的插件，以下是基本的创建和配置步骤：

  1. 创建一个文件夹，命名为插件的名称（可以与插件名称不同）。
  2. 在插件文件夹中创建一个名为 plugin.json 的文件，并将其放置在项目根目录。
  3. 配置 plugin.json 文件的基本结构，如下所示：

{
    "plugin_name": "name",
    "version": 1,
    "author": "by",
    "description": "description",
    "configuration": {
    }
}

在 plugin.json 文件中，您可以为插件设置名称、版本、作者、描述以及其他配置项。确保文件格式正确并包含必要的字段。

完成此配置后，您就能将插件安装到游戏中，并在 Option > Extensions 页面中看到它。

在你的插件项目根目录创建一个"A"文件夹(Assets)和"B"文件夹(Behaviors)来给插件添加功能

插件示例

以下是一个插件配置示例：


{
    "plugin_name": "Sword Extension",
    "version": 1.1,
    "author": "Realm of Echoes",
    "description": "Core extension",
    "configuration": {
        "item":[
            "Enhanced-Iron-Ingot",
            "Enhanced-Iron-Sword",
            "Geargrinder",
            "Star-Mist",
            "Unyielding-Forged-Blade",
            "Starlight",
            "Pure-Gold-Sword",
            "Crystal-Shard",
            "Platinum-Sword-II",
            "Chrome-longsword-II"
        ]
    }
}

在上面的示例中，configuration 中注册了插件所包含的物品（item）。您可以根据需要在配置中添加更多的功能项，甚至可以将 configuration 同时包含对象（object）和物品（item）列表。
